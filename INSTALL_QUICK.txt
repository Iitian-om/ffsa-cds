# ⚡ FFSA CDS - Installation Quick Reference

## 🎯 3-Minute Setup Guide

### Windows Users:

```
1. Open project folder
2. Right-click → Open in Terminal
3. Copy & Paste:
   Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope Process -Force; & .\install-nodejs.ps1
4. Wait 2-3 minutes
5. Restart VS Code/Terminal
6. Then run:
   pnpm install
   pnpm dev
7. Open: http://localhost:3000 🎉
```

## ✅ Verification Commands

```bash
# Check Node.js
node --version

# Check npm
npm --version

# Check pnpm
pnpm --version

# All should return version numbers like:
# v20.11.0
# 10.x.x
# 8.15.x
```

```bash

pnpm dev

# Open browser
http://localhost:3000
```

---

## 🆘 Troubleshooting

| Issue | Solution |
|-------|----------|
| **Command not found** | Restart terminal & use new window |
| **Installation fails** | Run as Administrator |
| **pnpm not found** | `npm install -g pnpm` |
| **Port 3000 in use** | `pnpm dev -- -p 3001` |
