## 🚀 **QUICK START — RIGHT NOW**

### **Windows Users (3 options):**

**Option 1 - PowerShell (Recommended):**
```
1. Open project folder in Terminal/PowerShell
2. Copy & Paste:
   Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope Process -Force; & .\install-nodejs.ps1
3. Wait 2-3 minutes
4. Restart VS Code
5. Run: pnpm install
6. Run: pnpm dev
7. Open: http://localhost:3000 ✅
```

**Option 2 - Double-Click:**
```
1. Double-click: setup.bat
2. Wait for completion
3. Restart VS Code
4. Run: pnpm install
5. Run: pnpm dev
6. Open: http://localhost:3000 ✅
```

**Option 3 - Manual:**
```
1. Download Node.js LTS from nodejs.org
2. Run installer
3. Restart terminal
4. Run: npm install -g pnpm
5. Follow Option 1 steps 5-7
```
